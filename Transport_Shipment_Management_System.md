# Transport & Shipment Management System

## Project Overview

বর্তমানে কোম্পানির Import ও Export-এর সকল তথ্য WhatsApp Group এবং Excel
Sheet-এর মাধ্যমে পরিচালনা করা হয়। প্রতিদিন কোন ট্রাক কোথায় যাচ্ছে, কোন পণ্য
বহন করছে এবং কোন ট্রাক গন্তব্যে পৌঁছেছে---এসব তথ্য ম্যানুয়ালি আপডেট করা হয়।

এই কাজগুলোকে আরও সহজ, দ্রুত এবং নির্ভুল করার জন্য একটি **Web-Based Management
System** তৈরি করা হবে।

## Project Goals

-   Import ও Export-এর সকল তথ্য এক জায়গায় সংরক্ষণ করা।
-   প্রতিটি ট্রাকের বর্তমান অবস্থা (Live Status) জানা।
-   কোন ট্রাকে কোন পণ্য যাচ্ছে তা সহজে দেখা।
-   Daily, Weekly ও Monthly Report এক ক্লিকে তৈরি করা।
-   WhatsApp-এর পরিবর্তে একটি কেন্দ্রীয় সিস্টেম ব্যবহার করা।

## Main Features

### 🚛 Truck Management

-   Truck Number
-   Company Truck / Outside Truck
-   Current Driver
-   Current Status
-   Current Location

### 👨‍✈️ Driver Management

-   Driver Name
-   Mobile Number
-   কোন Driver কোন Truck চালাচ্ছেন
-   Driver পরিবর্তনের History

### 📦 Shipment Management

-   Import / Export
-   Source
-   Destination
-   Truck
-   Driver
-   Product List
-   Quantity
-   Shipment Date

### 📍 Shipment Status

-   Loading
-   On The Way
-   Reached
-   Unloading
-   Completed

### 📊 Dashboard

-   আজ কতটি Export হয়েছে
-   আজ কতটি Import হয়েছে
-   কতটি Truck রাস্তায় আছে
-   কতটি Truck Loading হচ্ছে
-   কতটি Truck পৌঁছেছে
-   কতটি Shipment সম্পন্ন হয়েছে

### 📑 Reports

-   Daily Report
-   Weekly Report
-   Monthly Report
-   Truck Wise Report
-   Driver Wise Report
-   Product Wise Report

সব রিপোর্ট PDF বা Excel আকারে Print/Download করা যাবে।

## Expected Benefits

-   WhatsApp-এ তথ্য খোঁজার প্রয়োজন হবে না।
-   সকল তথ্য একটি সিস্টেমে সংরক্ষিত থাকবে।
-   Truck ও Shipment সহজে Track করা যাবে।
-   রিপোর্ট তৈরি করতে সময় কম লাগবে।
-   Manual ভুল কমে যাবে।
-   কোম্পানির দৈনন্দিন কাজ আরও দ্রুত ও সহজ হবে।
